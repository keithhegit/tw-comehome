# Cloudflare Pages 构建配置说明

## 必需的构建设置

在 Cloudflare Pages 项目设置中，请确保以下配置：

### 构建配置
- **框架预设**: `None` (或 `Vite`，如果有的话)
- **构建命令**: `npm run build`
- **构建输出目录**: `dist`
- **Node.js 版本**: `18` 或 `20`
- **环境变量**: 无需额外环境变量

### 重要提示
1. **确保构建输出目录正确**：必须是 `dist`（与 `vite.config.js` 中的 `outDir` 一致）
2. **检查构建日志**：如果构建失败，请查看构建日志中的错误信息
3. **清除缓存**：如果问题持续，尝试在 Cloudflare Pages 中清除构建缓存并重新部署

### 验证构建
构建成功后，`dist/index.html` 应该包含编译后的 JavaScript 文件引用（例如 `/assets/index-[hash].js`），而不是 `/src/main.jsx`。

如果 `dist/index.html` 中仍然有 `/src/main.jsx`，说明构建没有正确执行。
