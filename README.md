# 有出息 - H5 響應式前端應用

一個基於 React.js 技術棧的 H5 響應式前端應用，專為台灣用戶設計的綜合性服務平台。

## 🌟 產品特色

### 核心功能
- **投誠憑證系統** - 身份認證與安全撤離功能
- **福利激勵系統** - 分層福利與病毒裂變
- **社交功能** - 匿名社區與交友匹配
- **電商平台** - 半價商品與生活服務
- **雙模式設計** - 標準模式與簡化模式

### 設計亮點
- **繁體中文界面** - 完全本地化設計
- **響應式設計** - 完美適配手機瀏覽器
- **無障礙設計** - 高對比度與大字體選項
- **高壓模式** - 緊急情況下的全屏憑證顯示

## 🚀 快速開始

### 安裝依賴
```bash
npm install
```

### 開發模式
```bash
npm run dev
```

### 構建生產版本
```bash
npm run build
```

### 預覽生產版本

## 📱 技術棧

- **前端框架**: React 18
- **構建工具**: Vite
- **樣式方案**: Tailwind CSS
- **路由管理**: React Router DOM
- **圖標庫**: Lucide React
- **二維碼生成**: qrcode

## 🎨 設計系統

### 色彩系統
- **主色調**: `#0057B7` (信賴藍)
- **中性色**: 完整的灰階色彩系統
- **語義色**: 成功、警告、錯誤狀態色

### 字體系統
- **字體家族**: Noto Sans TC (思源黑體)
- **字體層級**: 從 12px 到 48px 的完整字號體系
- **雙模式**: 標準模式 16pt，簡化模式 18pt+

### 間距系統
- **網格基礎**: 8px 網格系統
- **圓角設計**: 8px, 12px, 16px 圓角規範
- **陰影系統**: 層次分明的陰影效果

## 🎯 雙模式設計

### 標準模式
- **目標用戶**: 年輕/中年用戶
- **導航**: 5個Tab（首頁、福利、交友、商城、社區）
- **字體**: 16pt 正文字號
- **功能**: 完整功能體驗

### 簡化模式
- **目標用戶**: 年長用戶/高壓場景
- **導航**: 4個Tab（首頁、投誠憑證、福利、我的）
- **字體**: 18pt+ 正文字號
- **設計**: 高對比度，大按鈕，簡化界面

## 📝 開發說明

### 本地開發
1. 確保 Node.js 版本 >= 16
2. 安裝依賴: `npm install`
3. 啟動開發服務器: `npm run dev`
4. 在瀏覽器中訪問 `http://localhost:3000`

### 代碼規範
- 使用 ESLint 進行代碼檢查
- 遵循 React Hooks 規範
- 組件命名使用 PascalCase
- 文件命名使用 camelCase

### 性能優化
- 組件懶加載
- 圖片優化
- 代碼分割
- 構建優化
